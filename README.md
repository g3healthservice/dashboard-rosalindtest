# RosalindTest × IBGE — Plataforma de Triagem Municipal

Dashboard estático (HTML/JS auto-contido) que cruza dados reais do **Censo 2022 do IBGE** (população total e mulheres 40+, por município — agregados SIDRA 4709 e 9514) com o modelo de ROI do **RosalindTest** (rastreio molecular de câncer de mama), para apoiar prefeituras na priorização e dimensionamento de programas de triagem.

- Seletor de município com busca (5.570 municípios do Brasil)
- Funil de rastreio (população-alvo → cobertura → exames → casos detectados → mortes evitadas) recalculado ao vivo
- ROI financeiro e socioeconômico
- Visão Comercial (pitch/CTA) e Visão Técnica (metodologia, fontes, enquadramento regulatório)

**Acesso online:** via GitHub Pages (ver link em Settings → Pages deste repositório).

Fontes: IBGE (Censo 2022), INCA (Dados e Números 2025), Observatório de Oncologia, material técnico LiqSci/RosalindTest.

Ferramenta de apoio à decisão — não substitui avaliação técnica, jurídica ou clínica formal. RosalindTest é um exame de rastreio (screening), complementar à mamografia, não um diagnóstico.
